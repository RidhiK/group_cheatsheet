Our Incandescent Cheatsheet

Basic Terminal Commands

  • pwd = print working directory
  • ls = list directory contents
  • cd directory_name = change directory to directory_name
  • cd .. = back up one folder
  • mkdir directory_name = make directory called directory_name

Github Commands

  • git status = check git status
  • git init = initialise git directory
  • git add --all = add all changes to staging area, ready to be committed
  • git reset --all = remove all changes from staging area
  • git commit -m "message" = commit changes, with description "message"
  • git push = push to github
  • git pull = pull shared directory changes from github
  • gitk -all = track changes of shared directory in wish
